# Custom 16-Bit Microcoded Computer

## Overview

This project presents a complete 16-bit computer designed from scratch using Logisim Evolution.

The system includes:

- Custom 16-bit CPU
- Microcoded Control Unit
- Custom Instruction Set Architecture (ISA)
- Stack-based Subroutine Support
- Python Assembler
- Assembly Operating System
- Keyboard Interface
- TTY Display
- Graphics Display Support
- Video Memory System
